Fauna Theme for Wordpress 1.0 (Final)
-------------------------------------
http://www.noscope.com/fauna/


Installation instructions
-------------------------
1. Download the zip-file, extract it
2. Place the "fauna" folder in your wp-content/themes/ folder
3. Go to your Wordpress admin -> Presentation, and activate the Fauna theme.


Changes
-------
See changelog.txt


Upgrading
---------
If you're upgrading from anything prior to Fauna RC2, a clean update is recommended. 


Installing Fauna in your language
---------------------------------
For now, Fauna is only translated into German. The german language file is located in the "languages" directory.

For more information on installing a language, please read:
http://codex.wordpress.org/Installing_WordPress_in_Your_Language

If you want to translate Fauna to your language, please read:
http://codex.wordpress.org/Translating_WordPress
Fauna comes with bundled "PO" and "POT" files, so it should be a breeze.
Note: If you need an RTL (Right-to-left) language, please read:
http://www.noscope.com/fauna/documentation/rtl/
